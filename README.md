# Cyber-Security-Threat-Analysis-Dashboard
Overview

The Cybersecurity Threat Analysis Dashboard is an interactive Power BI project designed to analyze global cybersecurity incidents and transform complex threat data into actionable insights. The dashboard provides a comprehensive overview of the cyber threat landscape, highlighting attack volumes, financial impact, affected users, resolution efficiency, attack categories, and year-over-year trends.

Using dynamic KPI cards, trend charts, distribution visualizations, and interactive filters, users can monitor the evolution of cyber threats, assess their global impact, identify dominant attack types, and support data-driven cybersecurity strategies and risk management decisions.
